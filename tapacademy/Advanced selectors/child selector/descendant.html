Full Stack Web Development

Full stack web development is the process of designing, creating, testing, and deploying a complete web application. It involves working with various technologies and tools, including:

  • Front-end Technologies
  • Back-end Technologies
  • Data bases

Front-end technologies

It is the visible part of website or web application which is responsible for user experience. The user directly interacts with the front end portion of the web application or website

  • HTML
  • CSS
  • JavaScript
